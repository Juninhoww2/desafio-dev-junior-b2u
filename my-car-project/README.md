## Instalação


primeiro rode a api na pasta api 

```bash
  cd api
  npm install
  npm run dev  
```

Instale my-project com npm

```bash
  npm install
  cd my-project
  npm run start 
```

pronto para abrir o projeto no frontend: localhost:3001
